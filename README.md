# 8 Ball Pool Game 🎱

A fully functional 8 Ball Pool game built with HTML5, CSS3, and JavaScript!

## Features

✅ **Full Game Mechanics**
- Realistic ball physics with collision detection
- Cue ball aiming and shooting system
- Power control for shot strength
- Pocket detection and ball removal
- Player turns and game win conditions

✅ **Visual Design**
- Beautiful green pool table
- Detailed ball rendering with colors and stripes
- Aiming guide lines
- Power meter display
- Real-time game statistics

✅ **Gameplay**
- 2-player multiplayer turns
- Solid vs Stripe ball types
- 8-ball win condition
- Game reset functionality

## How to Play

1. **Open** `pool.html` in your web browser
2. **Aim** by clicking and dragging on the white (cue) ball
3. **Adjust Power** by clicking the "+Power" button (observe the power meter)
4. **Shoot** by clicking the "SHOOT" button
5. **Win** by pocketing all your balls and then the 8 ball

## Controls

- **Left Mouse Click & Drag**: Aim the cue ball
- **+Power Button**: Increase shot power
- **SHOOT Button**: Execute the shot
- **New Game Button**: Start a fresh game

## Game Rules

- **Solids vs Stripes**: First player to pocket a ball gets that type
- **Pocket All Your Balls**: Before attempting the 8 ball
- **Pocket the 8 Ball Last**: After all your balls are pocketed to win
- **Lose Condition**: Pocketing the 8 ball before all your balls are gone

## Physics

The game includes:
- Ball velocity and momentum
- Friction simulation
- Collision detection between balls
- Wall bounce physics with energy loss
- Pocket detection and ball removal

- `pool.html` - Main HTML structure
- `pool-style.css` - Game styling and UI
