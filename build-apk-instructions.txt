⚙️ INSTALLATION STEPS ON ANDROID PHONE
─────────────────────────────────────────

1. Settings → Security → Enable "Unknown sources"
2. Download APK file from computer
3. Transfer to phone via:
   - USB cable (copy to Downloads folder)
   - Google Drive (download from cloud)
   - Email (open attachment)
4. Open file manager → Downloads
5. Tap APK file
6. Click "Install"
7. Wait for installation
8. App appears on home screen

💻 ADVANCED: ANDROID STUDIO METHOD
─────────────────────────────────────────

Prerequisites:
- Install Android Studio
- Install Java JDK
- Set up Android SDK

Steps:
1. Open Android Studio
2. Click "Open" → Select flutter-app folder
3. Wait for Gradle sync
4. Go to: Build → Generate Signed Bundle / APK
5. Select APK
6. Create or select signing key
7. Click Next → Release → Finish
8. Find APK at: app/release/app-release.apk

═══════════════════════════════════════════════════════════

🛠️ BUILD WITH FLUTTER CLI
─────────────────────────────────────────

1. Install Flutter: https://flutter.dev/docs/get-started/install
2. Clone repository
3. Open terminal in project folder
4. Run: flutter build apk --release
5. APK will be at: build/app/outputs/flutter-apk/app-release.apk

🐛 TROUBLESHOOTING
─────────────────────────────────────────

❌ "App not installed"
✅ Enable "Unknown sources" in Settings → Security

❌ "File not opening"
✅ Use a file manager app to navigate to file

❌ "Game won't load"
✅ Check internet connection
✅ Clear app cache and reinstall

❌ "Not enough storage"
✅ Free up 50MB space
